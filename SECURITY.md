# Security Policy

## Supported Versions

The following versions of Restoration-ai are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Restoration-ai, please report it responsibly.

**How to report:**

- Email your findings to the repository maintainer via the contact information listed on their GitHub profile.
- Please do **not** open a public GitHub issue for security vulnerabilities.

**What to expect:**

- You will receive an acknowledgment of your report within **48 hours**.
- We will investigate the issue and provide an update within **7 days**.
- If the vulnerability is confirmed, we will work on a fix and aim to release a patch as soon as possible.
- If the vulnerability is declined, we will provide an explanation.

**Disclosure:**

- We ask that you give us reasonable time to address the vulnerability before disclosing it publicly.
- We appreciate your help in keeping Restoration-ai and its users safe.
